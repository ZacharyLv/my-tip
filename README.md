# my-tip
使用electron开发的桌面程序，用于在固定时间后弹出系统提示，告诉用户要切换工作姿势了(站起来/坐下去)。
* 支持晨会时间的提示，语音提示
* 站立时间和坐下的时间可以分开设置
* 支持午休时间不弹出提示
* 支持下班时间提示
